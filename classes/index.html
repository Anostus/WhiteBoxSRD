Class Index

White Box has four possible character classes: the Cleric, the Fighter, the Magic-user, and the optional Thief. This site also includes an optional classes reference for referee-approved alternatives. Each class now has its own table-ready reference page.

The Cleric

Armored priests who serve a particular alignment, religion, or patron deity.

The Fighter

A warrior trained for battle and in the use of armor and weapons.

The Magic-User

A mysterious student of arcane powers and dark magic.

The Thief

An optional class relying on guile, stealth, and opportunistic violence.

Optional Classes

Bard, Assassin, Druid, Monk, Necromancer, Paladin, Ranger, and Swashbuckler options with advancement tables.
