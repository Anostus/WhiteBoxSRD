# Optional Classes Paladin Spell Progression Update

Updated `classes/optional-classes.html` to add the Paladin’s one-level-behind Cleric spell progression directly to the Paladin Advancement table.

## Changes

- Added `Spell 1` through `Spell 5` columns to the Paladin Advancement table.
- Populated Paladin spell slots by using the White Box Cleric advancement table one level behind:
  - Paladin 1: no effective Cleric level.
  - Paladin 2: casts as Cleric 1.
  - Paladin 3: casts as Cleric 2, and so on.
- Updated the Paladin feature note so it points to the expanded table rather than only telling the reader to consult the Cleric table.
