# Optional Classes Revision Report

Updated `classes/optional-classes.html` to streamline the optional-class rules presentation.

## Changes made

- Removed the separate **Rules notes** subsections from each optional class.
- Folded needed clarifications into **New or changed class features** where they directly explain a feature:
  - Bard daily spell use is now identified as class abilities.
  - Assassin Death Attack now explains Back Stab eligibility, hit requirement, and saving throw consequence.
  - Druid animal-only Charm Person now notes that normal spell saving throw and duration rules still apply.
  - Necromancer Banish/Turn Undead now explains the roll, control duration, `D` results, and no-effect dash results.
  - Paladin spellcasting now explains the one-level delay with examples.
  - Ranger favored-enemy bonus now specifies damage, not attack rolls.
  - Swashbuckler attack progression now specifies melee uses Fighter progression and ranged uses Thief progression.
- Added a Necromancer Banish/Control Undead table adapted from the Cleric Turn Undead table, with linked example undead.
- Expanded the Swashbuckler Advancement table with **To Hit (Melee)** and **To Hit (Ranged)** columns.
- Preserved and added internal links where appropriate for classes, spells, combat rules, equipment, and monsters.
